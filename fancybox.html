ZIP gallery for fancybox

ZipGallery is based on PHP code to extract a ZIP archive containing images of a gallery in conjunction with an AJAX driven front end. The image gallery itself is based on the fancybox jQuery plugin.

Click at one of the gallery links to see how things work!

  • Test gallery one try "gallery one" localised en
  • 2nd test gallery
  • 1st gallery again

Please note:

  • The first link "Test gallery one" utilises standard caption and thumbnail size defined in JavaScript file zipGallery-fancybox.js:
      dfltThumbSize = [50, 50];
      dfltCaption = '{%localised%&ensp;}{<b class="dim">&copy;%copyright%} - %filename%</b>';
  • The second link has custom defined caption and thumbnail size:
      data-thumbsize="75x50"
      data-caption="{%title%&ensp;}{<b class='dim'>&copy;%copyright%</b>}"
  • The "1st gallery again" link sets a colour attribute to parts of the caption:
      data-caption="%localised% <b style='color: #ff8800;'>&copy;%copyright%</b>"
